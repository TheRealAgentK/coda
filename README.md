# Coda

## A collection of date helper functions

### Methods

#### `formatDistance`

Returns the distance between two dates expressed as an approximate human-readable string

```
coda.formatDistance( "2021-01-01 10:00:00", "2021-01-01 11:05:23" );
// "about an hour"
```
